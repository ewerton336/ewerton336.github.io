## 🖥️ Sobre o portfólio

Este é o meu portfólio pessoal, **reimaginado no estilo Windows XP (tema Luna)** — com papel de parede Bliss, janelas arrastáveis, menu Iniciar, barra de tarefas e até a janela de contato como o **MSN Messenger**. É uma página única, 100% estática, sem frameworks nem dependências externas.

**🔗 Acesse:** https://ewerton336.github.io

### Recursos
- 🪟 Janelas de verdade: abrir, minimizar, maximizar, fechar e arrastar
- 🟢 Menu Iniciar em duas colunas e barra de tarefas fiéis ao XP
- 💬 Janela de Contato no estilo **MSN Messenger** (com os links como "contatos")
- 🌐 **Multi-idioma** (Português / English) com seletor na bandeja do sistema
- 📱 Totalmente responsivo — no celular cada janela vira tela cheia
- ⚡ HTML + CSS + JS puro (vanilla), sem build e sem dependências
